Проект включает в себя комплексный анализ данных (EDA) платформы Disney+ и построение моделей машинного обучения для решения задач классификации и регрессии.

## Этапы реализации проекта

### 1. Предобработка данных и Feature Engineering
* Заполнение пропусков в признаках `director`, `cast`, `country` и `rating` на основе логических группировок.
* Выделение числовых признаков `year_added` и `month_added` из текстовых дат.
* Создание признаков `is_festive_season` (флаг праздничного сезона) и `cast_count` (количество актеров).
* Оценка мультиколлинеарности через VIF (Variance Inflation Factor).

### ЧАСТЬ 1: Классификация (Тип контента)
**Задача:** Определение типа контента (Фильм / Сериал) на основе метаданных.
* **Модели:** `LogisticRegression`, `RandomForestClassifier`.
* **Результат:** Наилучшую точность показал Случайный Лес, успешно обработавший дисбаланс классов.

### ЧАСТЬ 2: Регрессия (Хронометраж фильмов)
**Задача:** Предсказание длительности фильмов в минутах.
* **Предобработка:** Фильтрация по `type == 'Movie'`, извлечение числового значения из строки длительности, применение One-Hot Encoding для жанров (250 признаков).
* **Модель:** `RandomForestRegressor` (criterion='absolute_error').
* **Метрики качества (Test score):**
  * MAE: 16.73 мин.
  * RMSE: 25.09 мин.
  * R²: 0.6327
