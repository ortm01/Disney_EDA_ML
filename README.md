Аналитический пайплайн и ML-моделирование для Disney+
Комплексный проект по анализу данных платформы Disney+: от глубокой очистки и восстановления пропусков до feature engineering и построения моделей машинного обучения для предсказания типа контента.

Архитектура проекта и функции
Весь код рефакторизован в модульную структуру на функциях:

load_clean_data() — загрузка данных и восстановление пропусков в режиссерах и странах на основе пересечения актерских составов (set) и группировок по жанрам.

coding_columns() — кодирование категориальных признаков и Feature Engineering (выделение года/месяца, флага праздничного сезона is_festive_season и подсчет размера каста).

vif_run() — проверка факторов на мультиколлинеарность через Variance Inflation Factor.

start_learn() — разбиение выборки, обучение моделей и валидация результатов.

Изменение бизнес-задачи
Изначально планировалось предсказывать возрастной рейтинг (rating), однако метрики уперлись в ~40%.

Вывод: Возрастной ценз привязан к содержанию фильма (сцены, диалоги), а не к косвенным признакам вроде года релиза или количества актеров.

Решение: Задача перестроена на предсказание типа контента (Movie / TV Show). Это логически оправдано, так как формат производства фильмов и сериалов (структура каста, рейтинговые системы США) принципиально отличается.

Результаты моделирования
Обучены две архитектуры: Логистическая регрессия и Случайный Лес (Random Forest). После смены целевой переменной точность выросла в два раза:

Logistic Regression Accuracy: 79.31%

Random Forest Accuracy: 80.69%

Метрики по классам (Classification Report):
Логистическая регрессия:
Movie: Precision = 0.83, Recall = 0.91

TV Show: Precision = 0.66, Recall = 0.49

Матрица ошибок: [[191, 20], [40, 39]]. Модель чаще ошибается в пользу большинства (фильмов), пропуская половину сериалов.

Случайный Лес:
Movie: Precision = 0.85, Recall = 0.89

TV Show: Precision = 0.66, Recall = 0.59

Матрица ошибок: [[187, 24], [32, 47]]. Ансамбль деревьев лучше справился с нелинейными связями и показал более высокий Recall на меньшем классе сериалов.
